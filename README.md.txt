# COVID-19 Data Analysis Using Python

## Project Overview
This project analyzes COVID-19 pandemic data to understand case progression, recovery trends, death trends, and country-level comparisons using Python.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Data Validation
4. Trend Analysis
5. Country Comparison
6. Data Visualization

## Key Analysis
- India COVID Case Trends
- 7-Day Rolling Average Analysis
- Death Trend Analysis
- Recovery Trend Analysis
- Active Case Analysis
- India vs USA vs China Comparison

## Visualizations
- Confirmed Cases Trend
- Rolling Average Trend
- Country Comparison Chart
- Death Trend
- Recovery Trend
- Active Cases Trend

## Key Insights
- COVID-19 cases increased rapidly during outbreak periods.
- Rolling averages provided smoother trend analysis.
- USA recorded higher confirmed cases than India and China.
- Recovery rates improved steadily over time.
- Active cases fluctuated significantly during peak waves.
